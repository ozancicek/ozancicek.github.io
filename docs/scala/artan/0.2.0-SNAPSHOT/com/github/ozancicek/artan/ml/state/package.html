    Key of the state

    stateIndex

    index of the filter, incremented only on state evolution

    state

    x_k, the state vector with length n_state

    stateCovariance

    state covariance matrix with dimensions n_state, n_state

    residual

    residual of x_k and z_k, vector with length n_obs

    residualCovariance

    covariance of residual, matrix with dimensions n_obs, n_obs

    eventTime

    event time of input measurement

    processModel

    processNoise

    measurementModel

    measurement model used at this state

    slidingLikelihood

    likelihood of consecutive measurements in a sliding window at this state
