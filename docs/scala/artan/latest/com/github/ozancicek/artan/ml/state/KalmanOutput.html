case class KalmanOutput(stateKey: String, stateIndex: Long, state: Vector, stateCovariance: Matrix, residual: Option[Vector], residualCovariance: Option[Matrix], eventTime: Option[Timestamp], processModel: Option[Matrix], processNoise: Option[Matrix], measurementModel: Option[Matrix], slidingLikelihood: Option[Double]) extends KeyedOutput[String] with Product with Serializable

Case class for representing the output state of a kalman filter. Let state at step k be denoted with x_k, measurement with z_k, where x_k and z_k are vectors of lenght n_state and n_obs

stateKey

Key of the state

stateIndex

index of the filter, incremented only on state evolution

state

x_k, the state vector with length n_state

stateCovariance

state covariance matrix with dimensions n_state, n_state

residual

residual of x_k and z_k, vector with length n_obs

residualCovariance

covariance of residual, matrix with dimensions n_obs, n_obs

eventTime

event time of input measurement

processModel

process model used at this state

processNoise

process noise used at this state

measurementModel

measurement model used at this state

slidingLikelihood

likelihood of consecutive measurements in a sliding window at this state
